# Fullstack User Dashboard

A fullstack React application that works with a REST API using JSON Server.

## Features
- User login & register
- Todos management
- Posts & comments
- Albums & photos

## Tech Stack
- React (Hooks, Router)
- REST API (Fetch, Async/Await)
- JSON Server

## Run the project

### client
npm install
npm run dev

### server
json-server --watch db.json --port 3001
